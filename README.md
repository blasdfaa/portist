# portist

Find which process is holding a port and free it in one click — right from your menu bar.

</div>

## Features

- 🟢 List of listening ports (TCP `LISTEN` and UDP) with auto-refresh
- 🗂 Grouped: databases · development · system · other
- 🔎 Search by port number or process name
- 📋 Process details: PID, path, command line, project, CPU, memory, uptime
- ↗ Open `localhost:port` in the browser with one click
- ✕ Kill your own process by PID
- 🎨 Theme: auto · light · dark
- 🚀 Launch at login and auto-update
